The framework is a set of standardized application programming interfaces (APIs), bound by a set of principles, that define fundamental software services to be consumed by business applications, which would be developed to satisfy Membersâ€™ terminology tooling use cases.

This mini web site introduces the concept of the IHTSDO Open Tooling Framework as well as the principles and a high level view of what the architectural components are.

All key links to other useful information on the progress and development can be found on the IHTSDO Tooling Services site page.

This information is being provided for developers, tooling software providers and IHTSDO Members and their users, to demonstrate the evolving and open approach to terminology tooling across the IHTSDO.
