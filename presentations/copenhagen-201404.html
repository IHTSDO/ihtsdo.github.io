IHTSDO SNOMED CT Browser

Beta release - http://browser.ihtsdotools.org

Browse SNOMED CT snapshot releases
Online, easily deployed, fully open source
Multi-tenant (International, Spanish, Danish & Swedish)
Multi-lingual user interface (English, Spanish & Danish)
User tour
Using NoSQL data storage

IHTSDO Refset Management Service

Beta release - https://refset.ihtsdotools.org

Create refset
Simple addition and removal of members
Search SNOMED using free text search
Import & export RF2 (snapshots only)
List, paginate, filter members/refsets/versions/releases
Version control like features
Create a "commit" (i.e. new version)
"Tag" a "commit" (i.e. create release)
Authentication
Security / permissions (who can do what)

IHTSDO Mapping Service

Beta release - http://mapping.ihtsdotools.org

Replacement for current desktop tool (Cartographer)
Viewing and editing of ICD9, ICD10 and FP/GP ICPC maps
Assigning batches to mappers
Manage conflicts
Workflow for approval ready for publication
